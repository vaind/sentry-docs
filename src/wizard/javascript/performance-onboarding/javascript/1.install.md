#### Install

Install our JavaScript browser SDK using either `yarn` or `npm`:

```bash {tabTitle: ESM}
# Using yarn
yarn add @sentry/browser @sentry/tracing
# Using npm
npm install --save @sentry/browser @sentry/tracing
```
